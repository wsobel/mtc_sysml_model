# Raw Material Package

This section provides semantic information for the [`RawMaterial`]({% link model/AssetInformationModel/RawMaterial/RawMaterial.md %}) model.

![RawMaterial]({% link figures/mtconnect.png %} "RawMaterial"){: width="50.0%"}

> Note: See *Section RawMaterial Schema Diagrams* for XML schema.

**Raw Material Asset Information Model**

Raw material represents the source of material for immediate use and sources of material that may or may not be used during the manufacturing process.

The [`RawMaterial`]({% link model/AssetInformationModel/RawMaterial/RawMaterial.md %}) [`Asset`]({% link model/AssetInformationModel/Asset.md %}) holds the references to the content stored in the actual [`RawMaterial`]({% link model/AssetInformationModel/RawMaterial/RawMaterial.md %}) container or derived about the [`RawMaterial`]({% link model/AssetInformationModel/RawMaterial/RawMaterial.md %}) by the system during operation.
